Description

Although our stopwatch can run pretty much infinitely, what we really want is to track specific time intervals. Let's set up a stopwatch with a user-defined upper limit that can be changed according to the user's needs.

To add the upper-limit functionality, implement a special pop-up dialog using DialogFragment.

Objectives

Create a new button with the text attribute Settings and the id attribute settingsButton. The Settings button is enabled only when the stopwatch is not running. After the button is clicked, DialogFragment with the following components should appear:

  1. EditText with the id attribute upperLimit;
  2. Default dialog buttons with the text attributes OK and Cancel.

If there is a number in EditText and the user clicks the OK button, the upper limit is set to that number representing time in seconds. If the user clicks Cancel, the dialog closes.

When the upper limit is reached, the stopwatch color is changed to red (use the RED constant of the class Color). If the number is not set, then there is no upper limit and the stopwatch behaves as in the previous stage.
